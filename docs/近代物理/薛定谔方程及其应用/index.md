# 薛定谔方程及其应用

薛定谔方程是量子力学的基本方程，描述了微观粒子的运动规律。

## 📚 本章内容

### [薛定谔方程](./薛定谔方程/)
量子力学的基本方程，波函数的概念和物理意义。

### [原子中的电子](./原子中的电子/)
氢原子的量子力学处理，电子云和原子轨道的概念。

## 🎯 学习要点

1. **薛定谔方程**：理解量子力学的基本方程
2. **波函数**：掌握波函数的物理意义和归一化条件
3. **概率密度**：理解波函数模平方的概率解释
4. **量子数**：掌握描述原子状态的量子数
5. **原子轨道**：理解电子云和原子轨道的概念

## 📖 量子力学的数学框架

薛定谔方程建立了量子力学的完整数学框架：

- **波动方程**：将粒子的运动描述为波动现象
- **概率解释**：引入了量子力学的统计性质
- **算符理论**：建立了量子力学的算符形式
- **原子结构**：为现代原子理论提供了理论基础
